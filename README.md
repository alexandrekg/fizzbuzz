# Fizz Buzz Robot

### Rules
1. If num is multiple of 3: return fizz
2. If num is multiple of 5: return buzz
3. If num is multiple of 3 and 5: return fizzbuzz
4. If num is not multiple of 3 and 5: return num itself

```python
from robot import robot 

robot(9)     'fizz'
robot(10)    'buzz'
robot(45)    'fizzbuzz'
robot(5002)  '5002'
```
